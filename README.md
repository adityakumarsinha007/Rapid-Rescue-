# 🚗 Rapid Rescue – IoT Accident Detection System

## 📌 Project Description
Rapid Rescue is an IoT-based autonomous accident detection and alert system designed for remote and low-connectivity areas. It detects vehicle accidents and instantly notifies rescue teams with real-time location.

---

## ⚙️ Working Flow
1. MPU6050 sensor detects sudden collision or abnormal motion  
2. ESP32 processes the sensor data  
3. Data is sent via Bluetooth to a web application  
4. Driver fills details and starts trip in Driver Mode  
5. Firebase stores and manages real-time data  
6. Rescue Dashboard displays alert with live GPS location using Leaflet map  

---

## 🛠 Tech Stack
- ESP32 Microcontroller  
- MPU6050 Sensor  
- Bluetooth Communication  
- Firebase (Realtime Database & API)  
- HTML, CSS, JavaScript  
- Leaflet.js (Map Integration)  

---

## 🚀 Features
- Real-time accident detection  
- Instant alert system  
- Live GPS tracking  
- Driver & Rescue interface  
- Works in low connectivity areas
